# Инструкция по установке и запуску

## Требования

- Node.js 18+ 
- npm 9+

## Установка зависимостей

```bash
npm install
```

## Запуск приложения

```bash
npm start
```

Приложение будет доступно по адресу: `http://localhost:4200`

## Сборка для production

```bash
npm run build
```

Собранные файлы будут в папке `dist/iot-sandbox`

## Структура проекта

```
src/
├── app/
│   ├── core/              # Основные сервисы, модели, store
│   │   ├── models/        # Модели данных
│   │   ├── services/      # Сервисы
│   │   └── store/         # NgRx store
│   ├── shared/            # Общие модули и компоненты
│   └── features/          # Функциональные модули
│       └── sandbox/       # Модуль песочницы
│           └── components/
│               ├── device-panel/          # Панель устройств
│               ├── workspace/             # Рабочая область
│               ├── device-card/           # Карточка устройства
│               ├── device-settings-dialog/ # Диалог настроек
│               ├── simulation-panel/      # Панель симуляции
│               ├── config-form/           # Форма конфигурации
│               └── template-selector/     # Селектор шаблонов
├── assets/                # Статические ресурсы
└── i18n/                  # Файлы переводов
```

## Основные возможности

1. **Drag & Drop устройств** - перетаскивание устройств из панели в рабочую область
2. **Настройка пороговых значений** - клик по устройству открывает диалог настроек
3. **Симуляция условий** - слайдеры для изменения значений датчиков
4. **Визуальная обратная связь** - устройства подсвечиваются при активации
5. **Шаблоны конфигураций** - предустановленные шаблоны для быстрого старта
6. **Сохранение в localStorage** - автоматическое сохранение конфигурации
7. **Валидация** - проверка корректности конфигурации перед отправкой
8. **Мультиязычность** - поддержка русского, английского и армянского языков

## API интеграция

Для интеграции с реальным API измените URL в `src/app/core/services/config.service.ts`:

```typescript
private readonly API_URL = 'https://your-api-url.com/api/configurations';
```

## Дополнительные настройки

### Изменение темы Material

Измените тему в `angular.json`:
```json
"styles": [
  "src/styles.scss",
  "node_modules/@angular/material/prebuilt-themes/indigo-pink.css"
]
```

Доступные темы:
- indigo-pink
- deeppurple-amber
- pink-bluegrey
- purple-green
